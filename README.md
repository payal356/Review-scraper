# Review-scraper

This code extracts reviews of different products available on nykaa.com and stores them in database (MongoDB).

Libraries used :
        Beautiful Soup, 
        Selenium,
        Pymongo
        
Used Gecko Web Driver to provide the link between tests in Selenium and the Firefox Browser.
